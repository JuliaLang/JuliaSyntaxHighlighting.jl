JuliaSyntaxHighlighting.paren_type — Function
paren_type(k::Kind) -> (Int, Symbol)

Return a pair of values giving the change in nesting depth caused by the paren k (+1 or -1), as well as a symbol indicating the kind of parenthesis:

  • ( and ) are a paren
  • [ and ] are a bracket
  • { and } are a curly

Anything else is of type none, and produced a depth change of 0.

JuliaSyntaxHighlighting._hl_annotations — Function
_hl_annotations(content::AbstractString, ast::GreenNode) -> Vector{Tuple{UnitRange{Int}, Pair{Symbol, Any}}}

Generate a list of (range, annot) pairs for the given content and ast.

The range is a UnitRange{Int} that indexes into ctx.content and annot is a Pair of the form :face => <face>.

This is a small wrapper around _hl_annotations! for convenience.
